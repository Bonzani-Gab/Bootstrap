cosa serve Bootstrap? Bootstrap è un framework front-end che facilita la creazione di siti web responsive e mobile-first. Fornisce una serie di strumenti e componenti predefiniti per lo sviluppo di interfacce utente coerenti.

Cosa devi importare per creare una pagina web con Bootstrap? <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare Bootstrap sul tuo sito? Un CDN (Content Delivery Network) è un network di server distribuiti che forniscono contenuti web ai visitatori in base alla loro posizione geografica.I vantaggi includono tempi di caricamento più rapidi, riduzione del carico sul server principale e miglioramenti nella disponibilità e nell'affidabilità

A cosa serve questo attributo del tag meta? content="width=device-width, initial-scale=1" serve a garantire che la pagina web sia responsiva e si adatti alle dimensioni dello schermo del dispositivo utilizzato.

Che differenza c’è tra Container e Container Fluid? Container: ha una larghezza fissa che varia a seconda delle dimensioni dello schermo. Container Fluid: occupa sempre il 100% della larghezza disponibile, adattandosi a qualsiasi dimensione dello schermo.
